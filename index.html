📡 Twitch-Einbindung für Streamer

So bindest du dein Idle-Dorf als Overlay in OBS ein.

1. Die Adresse
🐉
bkinvestment.de/idle-stream-mini
Rein visuelles Overlay: zeigt live mit, welchen Drachen du gerade auf der normalen Website bekämpfst - Dorf/Drache/HP-Balken, kompakt und transparent. Gesteuert/geklickt wird ausschließlich auf der Hauptseite.
2. In OBS einrichten
1️⃣
Quellen → + → Browser-Quelle → neue Quelle erstellen.
2️⃣
Die Adresse oben als URL eintragen.
3️⃣
Breite/Höhe nach Geschmack anpassen, z. B. 900×350 - die Seite passt sich automatisch an.
4️⃣
Haken bei „Quelle herunterfahren, wenn nicht sichtbar" nicht setzen - sonst pausiert die Anzeige, sobald die Szene nicht aktiv ist.
5️⃣
Einmalig einloggen: OBS hat einen eigenen, von deinem normalen Browser getrennten Speicher. Rechtsklick auf die Quelle → Interagieren → dort mit deinem BKMP-Account (Ingame-Name + Passwort) anmelden. Danach bleibst du eingeloggt.
3. Fertig!
✅
Gekämpft/geklickt wird ganz normal auf der Hauptseite - das Overlay zeigt nur live mit, ohne dass man dort etwas anklicken kann.
📲 Auf den Startbildschirm holen, damit es sich wie eine echte App anfühlt.
